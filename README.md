# ilkservice

Продающий сайт сервиса по разборке и ремонту американских грузовиков в Ростовской области.

## Возможности

- адаптивная вёрстка для компьютеров и телефонов;
- направления ремонта и разборки;
- каталог услуг по б/у запчастям;
- форма заявки;
- оригинальный главный визуал.

## Локальный запуск

Откройте `index.html` в браузере. Сборка и установка зависимостей не требуются.

## Перед публикацией

- добавьте юридические реквизиты и страницу политики конфиденциальности.
- после публикации добавьте сайт в Яндекс Вебмастер и Google Search Console.

Сайт не требует сборки и внешнего JavaScript-фреймворка.

Форма формирует готовую заявку для отправки на `ilira72@mail.ru`.

В проекте настроены метатеги, canonical для `https://ilkservice.ru/`, Open Graph, локальная структурированная разметка `AutoRepair`, `robots.txt`, `sitemap.xml`, семантические заголовки и серверное сжатие/кэширование для Apache-совместимого хостинга.

Публичная тестовая версия автоматически разворачивается через GitHub Pages из ветки `main`.
