# ilkservice

Продающий сайт сервиса по разборке и ремонту американских грузовиков в Ростовской области.

## Возможности

- адаптивная вёрстка для компьютеров и телефонов;
- направления ремонта и разборки;
- каталог услуг по б/у запчастям;
- форма заявки;
- оригинальный главный визуал.

## Локальный запуск

Откройте `index.html` в браузере. Сборка и установка зависимостей не требуются.
